### P1 · 树.md — maxPathSum 代码语法错误（无法运行）

- 位置：树.md 第 327-331 行（二叉树中的最大路径和）
- 问题：
  ```python
  class Solution:
          # 既然有可能全是负数节点...
          self.max_sum = float('-inf')   # ← 直接写在类体内，缩进异常

      def maxPathSum(self, root):
  ```
  `self.max_sum` 被放在类体（class body）里，此处没有 `self` 上下文，也没有 `__init__`。这段代码会直接抛 `NameError: name 'self' is not defined`，类无法定义。
- 影响：本题是高频困难题，flagship 代码跑不起来会直接误导面试准备。
- 修复建议：把初始化移进方法内部，例如在 `maxPathSum` 函数体首行写 `self.max_sum = float('-inf')`，或新增 `__init__`。对照同文件 diameterOfBinaryTree（第 288-299 行）的写法即可。

- 链表.md：反转/环检测/环入口（Floyd 数学证明正确）、寻找重复数（数组转链表 + Floyd）、随机链表深拷贝（两趟哈希）——均正确。
- 树.md：直径（base case 返回 -1 的边数语义自洽，已验证叶子/三节点）、LCA、前序+中序建树、展开链表（逆后序头插）、路径总和 III（前缀和+回溯撤销）、验证 BST（区间限定）——逻辑均正确。**唯一缺陷是 maxPathSum 的语法错误（见 P1）**，其算法思路本身正确。
- 堆.md：快速选择（双路+三路荷兰国旗，对全等数组退化的分析准确）、桶排序求前 K 高频（O(N)）、对顶堆求中位数（平衡不变式已验证 add 1/2 次的尺寸演化，正确）——均正确。
- 字符串.md：滑动窗口三连（最长无重复/异位词/最小覆盖）、最长回文中心扩展（`2n-1` 中心 + `l=i//2, r=(i+1)//2` 映射推导正确）、字符串相乘（i+j / i+j+1 竖式定位正确）、最长有效括号（栈存下标 + -1 地基）——均正确。
