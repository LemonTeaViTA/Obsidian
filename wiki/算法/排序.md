## 一、排序核心思想与区间问题套路

面试中很少直接让你手写快排、归并（除非面试官特别要求），反而更多的是考察：
1. **区间合并类问题**：套路极强，**先按区间的左端点排序**！排完序后，能合并的区间必然是相邻的。
2. **特定的原地排序（荷兰国旗问题）**：只能扫描一趟，要求 `O(1)` 空间。这时候**三指针**就派上用场了。
3. **自定义排序规则**：给定一堆字符串或者数字，问怎么拼出最大/最小的数字，需要定义自己的 `cmp`。

## 二、区间操作

### 合并区间（难度：中等）

**题目：**
以数组 `intervals` 表示若干个区间的集合，其中单个区间为 `intervals[i] = [start_i, end_i]`。请你合并所有重叠的区间，并返回一个不重叠的区间数组。

**【模板解析与思路归纳】：**
- **必须先按左边界排序！**
- 排序后，我们搞一个结果数组 `merged`。把区间挨个往里面送。
- 如果当前区间 `curr` 的左边界 `>` 结果数组最后一个区间 `last` 的右边界，说明两人毫无交集，直接把 `curr` 塞入结果。
- 否则两人有交叠，既然左边界已经排过序不怕了，我们只要把 `last` 的右边界更新为两人右边界的**最大值**即可。

**答案代码（Python版）：**
```python
class Solution:
    def merge(self, intervals: List[List[int]]) -> List[List[int]]:
        if not intervals:
            return []
            
        # 按区间左端点升序排列
        intervals.sort(key=lambda x: x[0])
        merged = []
        
        for interval in intervals:
            # 如果结果数组为空，或者当前区间的左端点 > 结果数组最后一个区间的右端点，不需要合并
            if not merged or merged[-1][1] < interval[0]:
                merged.append(interval)
            else:
                # 否则说明有重合，延长结果数组末尾区间的右端点
                merged[-1][1] = max(merged[-1][1], interval[1])
                
        return merged
```

## 三、三指针与荷兰国旗

### 颜色分类（难度：中等）

**题目：**
给定一个包含红色、白色和蓝色，一共 `n` 个元素的数组 `nums`，原地对它们进行排序，使得相同颜色的元素相邻，并按照红色、白色、蓝色顺序排列。使用整数 0、 1 和 2 分别表示红色、白色和蓝色。要求只能扫描一遍。

**【模板解析与思路归纳】：**
- 经典的**荷兰国旗问题**。如果可以扫描两遍，那直接哈希表计数再重新赋值就行了。扫描一遍必须使用**三指针**：
  - `left` 指向下一个塞 `0` (红色) 的位置。
  - `right` 指向下一个塞 `2` (蓝色) 的位置。
  - `i` 作为随波逐流的探索者。
- 如果 `nums[i] == 0`，就把它换去前面 `left` 的位置，`left` 和 `i` 都前进一步。
- 如果 `nums[i] == 2`，就把它换去后面 `right` 的位置，`right` 后退一步（此时被换过来的数字还没检查，故 `i` 停在原地等待下一次检查）。
- 取 `1` 啥也不干，`i` 往前走。

**答案代码（Python版）：**
```python
class Solution:
    def sortColors(self, nums: List[int]) -> None:
        """
        Do not return anything, modify nums in-place instead.
        """
        left = 0
        right = len(nums) - 1
        i = 0
        
        while i <= right:
            if nums[i] == 0:
                nums[left], nums[i] = nums[i], nums[left]
                left += 1
                i += 1
            elif nums[i] == 2:
                nums[right], nums[i] = nums[i], nums[right]
                right -= 1
            else:
                i += 1
```
