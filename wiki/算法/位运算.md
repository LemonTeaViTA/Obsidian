## 一、位运算核心思想与归类技巧

位运算虽然在平时开发中用得不多，但在算法和底层源码中却极为常见。它的核心优势是**极致的性能**和**利用计算机底层的加解密特性**。
遇到位运算题目，牢记以下几大核心套路：
1. **异或 `^` 的三大天性**：
   - 任何数与自己异或为 0：`a ^ a = 0`
   - 任何数与 0 异或为自己：`a ^ 0 = a`
   - 满足交换律与结合律：`a ^ b ^ a = b` (非常适合解决“找出唯一不重复数字”问题)。
2. **经典 Trick：消除最低位的 1**：`n & (n - 1)`，在统计二进制中 1 的个数或者判断是否为 2 的幂时简直是量身定制。
3. **获取最低位的 1**：`n & (-n)`，树状数组（Fenwick Tree）的基石。

## 二、基础位运算技巧

### 只出现一次的数字（难度：简单）

**题目：**
给你一个非空整数数组 `nums`，除了某个元素只出现一次以外，其余每个元素均出现两次。 找出那个只出现了一次的元素。要求具有线性的时间复杂度并使用常数级额外空间。

**【模板解析与思路归纳】：**
- 要求 `O(n)` 并且额外空间 `O(1)`，也就是说连哈希表都不让用。
- 直接祭出**异或全家桶**。因为 `a ^ a = 0` 且满足交换律，所有出现两次的数字在互为异或的时候都会自爆成 `0`。
- 最终留下来的，就是那个唯一没有兄弟跟它同归于尽的数字。

**答案代码（Python版）：**
```python
class Solution:
    def singleNumber(self, nums: List[int]) -> int:
        res = 0
        for num in nums:
            res ^= num
        return res
```

### 位 1 的个数（难度：简单）

**题目：**
编写一个函数，输入是一个无符号整数（以二进制串的形式），返回其二进制表达式中数字位数为 `'1'` 的个数（也被称为汉明重量）。

**【模板解析与思路归纳】：**
- 新手做法：转成字符串数 `1`，或者每次向右移位然后 `& 1` 判断最后一位是不是 `1`。
- **模板做法**：巧用 `n & (n - 1)`。
  - 因为 `n - 1` 的本质就是把最低位的 `1` 变成 `0`，并且把这个 `1` 后面的所有 `0` 置为 `1`。
  - 那么 `n & (n - 1)`，刚才那个最低位的 `1` 及以后的所有位，刚好全部变成了 `0`！也就是“消除了最低位的 1”。

**答案代码（Python版）：**
```python
class Solution:
    def hammingWeight(self, n: int) -> int:
        count = 0
        while n:
            n &= (n - 1)  # 每次消灭最右边的一个 1
            count += 1
        return count
```
