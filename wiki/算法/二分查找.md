## 一、二分查找核心思想与终极模板

二分查找的核心思想不是在数组中“找元素”，而是**“在一个单调（或具备某种排他性两段结构）的区间内，通过检验中间条件，每次能稳定排除一半的不可能域”**。

### 🌟 三大经典模板框架
许多人写二分永远在纠结 `left <= right` 还是 `left < right`，返回 `left` 还是 `right`。只要死记以下基础和变型框架，即可通吃：

**1. 模板一：寻找精确值（最基础）**
*   **循环条件**：`while (left <= right)`（因为你要搜遍 `[left, right]` 这个闭区间）。
*   **退出时机**：找到直接 `return mid;`；找不到时 `left == right + 1`（区间为空退出）。
*   **区间缩减**：`left = mid + 1;` 且 `right = mid - 1;`。

**2. 模板二：寻找边界（左/右侧最早或最晚出现位置）**
*   **循环条件**：依然推荐 `while (left <= right)`，但在命中 `target` 时**绝不提前 return**！
*   **找左侧边界**：遇到相等的 `nums[mid] == target` 继续往左缩逼，即 `right = mid - 1;`。最后验证 `left` 所指是否等于 target。
*   **找右侧边界**：遇到相等的继续往右缩逼，即 `left = mid + 1;`。最后验证 `right` 所指。

*(注：计算 `mid` 时统一使用 `mid = left + (right - left) // 2`，以防止大数溢出，且在 Python 中保持整除)*

## 二、基础入门：标准二分与搜边界
*利用最基础的大小关系来排除一半边界。*

### 二分查找（难度：简单）

**题目：**
给定一个 n 个元素有序的（升序）整型数组 nums 和一个目标值 target，写一个函数搜索 nums 中的 target，如果目标值存在返回下标，否则返回 -1。

**【模板解析与思路归纳】：**
- 这是最标准的**【寻找精确值模板】**。
- 只要判断 `nums[mid] == target` 就立刻返回。
- 循环条件：`while (left <= right)`。
- 转移跳过：左侧偏小就去右侧找 `left = mid + 1`，偏大就去左侧找 `right = mid - 1`。

**答案代码（Python版）：**
```python
class Solution:
    def search(self, nums: List[int], target: int) -> int:
        left, right = 0, len(nums) - 1
        
        while left <= right:
            mid = left + (right - left) // 2
            if nums[mid] == target:
                return mid
            elif nums[mid] < target:
                left = mid + 1
            else:
                right = mid - 1
                
        return -1
```

### 搜索插入位置（难度：简单）

**题目：**
给定一个排序数组和一个目标值，在数组中找到目标值，并返回其索引。如果目标值不存在于数组中，返回它将会被按顺序插入的位置。
请必须使用时间复杂度为 `O(log n)` 的算法。

**【模板解析与思路归纳】：**
- 这题本质是**寻找第一个大于等于 target 的位置（左边界思想的雏形）**。
- 用基础精确值模板，但是当循环因为找不到而结束时（`left > right` 触发跳出），`left` 指针刚好会停在这个插入的地方（恰好越过 `right` 的那个位置，也就是插入的新位置）。

**答案代码（Python版）：**
```python
class Solution:
    def searchInsert(self, nums: List[int], target: int) -> int:
        left, right = 0, len(nums) - 1
        
        while left <= right:
            mid = left + (right - left) // 2
            if nums[mid] == target:
                # 找到了位置，如果是自己就插在自己这
                return mid
            elif nums[mid] < target:
                left = mid + 1
            else:
                right = mid - 1
                
        # 跳出循环时 left 的位置正好就是第一个比 target 大的数字的位置
        return left
```

### 在排序数组中查找元素的第一个和最后一个位置（难度：中等）

**题目：**
给你一个按照非递减顺序排列的整数数组 nums，和一个目标值 target。请你找出给定目标值在数组中的开始位置和结束位置。
如果数组中不存在目标值 target，返回 [-1, -1]。必须设计并实现时间复杂度为 `O(log n)` 的算法。

**【模板解析与思路归纳】：**
- 这是**最正统的【寻找左右边界】双模板齐放题**。
- **找第一位 (左边界)**：即使 `nums[mid] == target` 也别返回！而是收缩右逼，让 `right = mid - 1`，企图继续往左寻找更早出现的。
- **找最后一位 (右边界)**：同理，即使命中，也收缩左逼，让 `left = mid + 1`，企图继续往右逼近！
- 最后判定一次：我们收缩出来的边界那个元素，到底等不等于目标值？

**答案代码（Python版）：**
```python
class Solution:
    def searchRange(self, nums: List[int], target: int) -> List[int]:
        def findLeft():
            left, right = 0, len(nums) - 1
            res = -1
            while left <= right:
                mid = left + (right - left) // 2
                if nums[mid] == target:
                    res = mid
                    right = mid - 1  # 核心：遇到相等也把右眼盖住，往左逼近
                elif nums[mid] < target:
                    left = mid + 1
                else:
                    right = mid - 1
            return res

        def findRight():
            left, right = 0, len(nums) - 1
            res = -1
            while left <= right:
                mid = left + (right - left) // 2
                if nums[mid] == target:
                    res = mid
                    left = mid + 1   # 核心：遇到相等也把左眼盖住，往右逼近
                elif nums[mid] < target:
                    left = mid + 1
                else:
                    right = mid - 1
            return res

        return [findLeft(), findRight()]
```

## 三、局部有序/旋转数组模型

### 寻找旋转排序数组中的最小值（难度：中等）

**题目：**
已知一个长度为 n 的数组，预先按照升序排列，经由 1 到 n 次 旋转 后，得到输入数组。找里面最小的元素。必须在 `O(log n)` 复杂度内完成。

**【模板解析与思路归纳】：**
- 这题是旋转数组一切基础。如何二分？**拿 `mid` 的值和右端点 `right` 的值做比较**。
- 若 `nums[mid] < nums[right]`：说明右半边顺滑，是纯递增的。那么极小值只可能在左边那段混沌区，或者正好是 `mid` 自己！此时令 `right = mid` (不能跳过自身！)。
- 若 `nums[mid] > nums[right]`：说明右半边混沌，发生过断层，最小值必定在右半边，令 `left = mid + 1`。

**答案代码（Python版）：**
```python
class Solution:
    def findMin(self, nums: List[int]) -> int:
        left, right = 0, len(nums) - 1
        
        while left < right:
            mid = left + (right - left) // 2
            if nums[mid] > nums[right]:
                left = mid + 1
            else:
                right = mid
                
        return nums[left]
```

### 搜索旋转排序数组（难度：中等）

**答案代码（Python版）：**
```python
class Solution:
    def search(self, nums: List[int], target: int) -> int:
        left, right = 0, len(nums) - 1
        
        while left <= right:
            mid = left + (right - left) // 2
            if nums[mid] == target:
                return mid
            
            # 判断哪半边是纯纯的有序段
            if nums[left] <= nums[mid]:
                # 左半部分有序
                if nums[left] <= target < nums[mid]:
                    right = mid - 1
                else:
                    left = mid + 1
            else:
                # 右半部分一定是有序的
                if nums[mid] < target <= nums[right]:
                    left = mid + 1
                else:
                    right = mid - 1
                    
        return -1
```

## 四、二分答案模型（终极天花板）

### 寻找两个正序数组的中位数（难度：困难）

**答案代码（Python版）：**
```python
class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        def getKthElement(k: int) -> int:
            index1, index2 = 0, 0
            while True:
                # 边界1：当 nums1 已经被切完，直接取 nums2 的第 k 个
                if index1 == len(nums1):
                    return nums2[index2 + k - 1]
                if index2 == len(nums2):
                    return nums1[index1 + k - 1]
                # 边界2：当 k 收缩到了 1，说明就是这两个当前头部较小的那个了
                if k == 1:
                    return min(nums1[index1], nums2[index2])
                
                # 正常情况：两个数组各自往前探 k//2 的距离
                half = k // 2
                newIndex1 = min(index1 + half - 1, len(nums1) - 1)
                newIndex2 = min(index2 + half - 1, len(nums2) - 1)
                
                pivot1, pivot2 = nums1[newIndex1], nums2[newIndex2]
                if pivot1 <= pivot2:
                    # nums1 前区更小，砍掉并减去砍掉的数目
                    k -= (newIndex1 - index1 + 1)
                    index1 = newIndex1 + 1
                else:
                    k -= (newIndex2 - index2 + 1)
                    index2 = newIndex2 + 1

        totalLength = len(nums1) + len(nums2)
        if totalLength % 2 == 1:
            return float(getKthElement(totalLength // 2 + 1))
        else:
            return (getKthElement(totalLength // 2) + getKthElement(totalLength // 2 + 1)) / 2.0
```
