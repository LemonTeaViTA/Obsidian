## 一、二分查找核心思想与终极模板

二分查找的核心思想不是在数组中“找元素”，而是**“在一个单调（或具备某种排他性两段结构）的区间内，通过检验中间条件，每次能稳定排除一半的不可能域”**。

### 🌟 三大经典模板框架
许多人写二分永远在纠结 \`left <= right\` 还是 \`left < right\`，返回 \`left\` 还是 \`right\`。只要死记以下基础和变型框架，即可通吃：

**1. 模板一：寻找精确值（最基础）**
*   **循环条件**：\`while left <= right\`（因为你要搜遍 \`[left, right]\` 这个闭区间）。
*   **退出时机**：找到直接 \`return mid\`；找不到时 \`left == right + 1\`（区间为空退出）。
*   **区间缩减**：\`left = mid + 1\` 且 \`right = mid - 1\`。

**2. 模板二：寻找边界（左/右侧最早或最晚出现位置）**
*   **循环条件**：依然推荐 \`while left <= right\`，但在命中 \`target\` 时**绝不提前 return**！
*   **找左侧边界**：遇到相等的 \`nums[mid] == target\` 继续往左缩逼，即 \`right = mid - 1\`。最后验证 \`left\` 所指是否等于 target。
*   **找右侧边界**：遇到相等的继续往右缩逼，即 \`left = mid + 1\`。最后验证 \`right\` 所指。

*(注：计算 \`mid\` 时统一使用 \`mid = (left + right) // 2\`。在严谨的语言如C++/Java中为了防溢出写成 \`mid = left + (right - left) // 2\`)。*

## 二、基础入门：标准二分与搜边界
*利用最基础的大小关系来排除一半边界。*

### 二分查找（难度：简单）

**题目：**
给定一个 n 个元素有序的（升序）整型数组 nums 和一个目标值 target，写一个函数搜索 nums 中的 target，如果目标值存在返回下标，否则返回 -1。

**【模板解析与思路归纳】：**
- 这是最标准的**【寻找精确值模板】**。
- 只要判断 \`nums[mid] == target\` 就立刻返回。
- 循环条件：\`while left <= right\`。
- 转移跳过：左侧偏小就去右侧找 \`left = mid + 1\`，偏大就去左侧找 \`right = mid - 1\`。

**答案代码：**
\`\`\`python
def search(nums, target):
    left, right = 0, len(nums) - 1
    
    while left <= right:
        mid = (left + right) // 2
        if nums[mid] == target:
            return mid
        elif nums[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
            
    return -1
\`\`\`

### 搜索插入位置（难度：简单）

**题目：**
给定一个排序数组和一个目标值，在数组中找到目标值，并返回其索引。如果目标值不存在于数组中，返回它将会被按顺序插入的位置。
请必须使用时间复杂度为 \`O(log n)\` 的算法。

**【模板解析与思路归纳】：**
- 这题本质是**寻找第一个大于等于 target 的位置（左边界思想的雏形）**。
- 用基础精确值模板，但是当循环因为找不到而结束时（\`left > right\` 触发跳出），\`left\` 指针刚好会停在这个插入的地方（恰好越过 \`right\` 的那个位置，也就是插入的新位置）。

**答案代码：**
\`\`\`python
def searchInsert(nums, target):
    left, right = 0, len(nums) - 1
    
    while left <= right:
        mid = (left + right) // 2
        if nums[mid] == target:
            # 找到了位置，如果是自己就插在自己这
            return mid
        elif nums[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
            
    # 跳出循环时 left 的位置正好就是第一个比 target 大的数字的位置
    return left
\`\`\`

### 在排序数组中查找元素的第一个和最后一个位置（难度：中等）

**题目：**
给你一个按照非递减顺序排列的整数数组 nums，和一个目标值 target。请你找出给定目标值在数组中的开始位置和结束位置。
如果数组中不存在目标值 target，返回 [-1, -1]。必须设计并实现时间复杂度为 \`O(log n)\` 的算法。

**【模板解析与思路归纳】：**
- 这是**最正统的【寻找左右边界】双模板齐放题**。
- **找第一位 (左边界)**：即使 \`nums[mid] == target\` 也别返回！而是收缩右逼，让 \`right = mid - 1\`，企图继续往左寻找更早出现的。
- **找最后一位 (右边界)**：同理，即使命中，也收缩左逼，让 \`left = mid + 1\`，企图继续往右逼近！
- 最后判定一次：我们收缩出来的边界那个元素，到底等不等于目标值？

**答案代码：**
\`\`\`python
def searchRange(nums, target):
    def findLeft(nums, target):
        left, right = 0, len(nums) - 1
        while left <= right:
            mid = (left + right) // 2
            if nums[mid] < target:
                left = mid + 1
            else:
                # 核心：遇到相等也把右眼盖住往左逼近
                right = mid - 1
        # 跳出后 left 有可能越界，或者 left 指向的根本不是 target
        if left >= len(nums) or nums[left] != target:
            return -1
        return left

    def findRight(nums, target):
        left, right = 0, len(nums) - 1
        while left <= right:
            mid = (left + right) // 2
            if nums[mid] <= target:
                # 核心：遇到相等也把左眼盖住往右逼近
                left = mid + 1
            else:
                right = mid - 1
        # 找右边界结束时，落在 right 身上
        if right < 0 or nums[right] != target:
            return -1
        return right

    return [findLeft(nums, target), findRight(nums, target)]
\`\`\`

## 三、局部有序/旋转数组模型
*当数组只有局部单调时（比如被旋转剪切了两半），二分的精华依然有效：只要随便砍一刀，必定有一半是单调的，就在那一半里面做判断抛弃即可。*

### 寻找旋转排序数组中的最小值（难度：中等）

**题目：**
已知一个长度为 n 的数组，预先按照升序排列，经由 1 到 n 次 旋转 后，得到输入数组。例如，原数组 nums = [0,1,2,4,5,6,7] 分为两截颠倒位置后变成了 [4,5,6,7,0,1,2]。找里面最小的元素。必须在 \`O(log n)\` 复杂度内完成。

**【模板解析与思路归纳】：**
- 这题是旋转数组一切基础。如何二分？**拿 \`mid\` 的值和右端点 \`right\` 的值做比较**。
- 若 \`nums[mid] < nums[right]\`：说明右半边顺滑，是纯递增的。那么极小值只可能在左边那段混沌区，或者正好是 \`mid\` 自己！此时令 \`right = mid\` (注意不是 mid-1，不能跳过自身！)。
- 若 \`nums[mid] > nums[right]\`：说明右半边混沌，发生过断层，最小值必定在右边那截里面，左半边是爬坡的。令 \`left = mid + 1\` 即可。

**答案代码：**
\`\`\`python
def findMin(nums):
    left, right = 0, len(nums) - 1
    
    # 只要是不找绝对目标，而是在找极值点，一般用 left < right
    while left < right:
        mid = (left + right) // 2
        if nums[mid] > nums[right]:
            # mid > right，最小值在右半边的断层里
            left = mid + 1
        else:
            # 这边是有序上坡的，最小值必定在左边（且包含了mid自己）
            right = mid
            
    # 最后收缩到一个点
    return nums[left]
\`\`\`

### 搜索旋转排序数组（难度：中等）

**题目：**
整数数组 nums 按升序排列且无重复。在某个未知的下标上进行了旋转。给你一个整数 target，如果 nums 中存在这个目标值 target 返回下标，否则返回 -1。

**【模板解析与思路归纳】：**
- **这是经典的二分旋转板子**。我们要找具体值，所以恢复使用 \`left <= right\` 模板。
- 逻辑：无论怎么把一个递增序列切一刀，它的左右两半里，**必定有一半是完全纯递增的**！
- 如果 \`nums[left] <= nums[mid]\`，说明 **左半区是纯递增的**。接着只要判断 \`target\` 是不是稳稳落在这个左边闭眼里，如果是：\`right = mid - 1\`，否则就去另一半找：\`left = mid + 1\`。
- 反之说明 **右半区是纯递增的**。看 \`target\` 在不在右区间即可。

**答案代码：**
\`\`\`python
def search(nums, target):
    left, right = 0, len(nums) - 1
    
    while left <= right:
        mid = (left + right) // 2
        if nums[mid] == target:
            return mid
            
        # 判断哪半边是纯纯的有序段
        if nums[left] <= nums[mid]:
            # 左半部分有序
            if nums[left] <= target < nums[mid]: # 判断目标在不在这截有序区内
                right = mid - 1
            else:
                left = mid + 1
        else:
            # 右半部分一定是有序的
            if nums[mid] < target < nums[right]: # 同理判断靶子是否在右截里
                left = mid + 1
            else:
                right = mid - 1
                
    return -1
\`\`\`

## 四、二分答案模型（终极天花板）
*在普通方法想破脑壳时，二分最高阶的用法是——不再是对数组下标进行二分，而是直接在【解的空间】里对答案进行二分检查！*

### 寻找两个正序数组的中位数（难度：困难）

**题目：**
给定两个大小分别为 m 和 n 的正序（从小到大）数组 nums1 和 nums2。请你找出并返回这两个正序数组的中位数。时间复杂度必须是 O(log (m+n))。

**【模板解析与思路归纳】：**
- 既然要合并找中位数，本质上是**找两个合并后第 K 小的数字（K为总长度一半）**。
- 不再是对数组索引漫无目的的二分，而是根据需要切除的数目 \`K/2\`，去比较两个数组各自伸出去 \`K/2\` 处的大小。
- **排他律**：谁在那里的元素小，谁的 \`前 K/2\` 个元素就绝对不可能成为第 K 大的数（它们连塞牙缝都不够），统统大胆抛弃去缩小查找范围！这就非常优美地完成了“折半删除”。

**答案代码：**
\`\`\`python
def findMedianSortedArrays(nums1, nums2):
    def getKthElement(k):
        """
        在 nums1 和 nums2 中寻找第 k 小的元素（1-based）
        """
        index1, index2 = 0, 0
        while True:
            # 边界1：当 nums1 为空（或者被切完了），直接返回 nums2 加上 k 偏移量
            if index1 == len(nums1):
                return nums2[index2 + k - 1]
            if index2 == len(nums2):
                return nums1[index1 + k - 1]
            # 边界2：当 k 收缩到了 1，说明就是这两个当前头部较小的那个了
            if k == 1:
                return min(nums1[index1], nums2[index2])

            # 正常情况：两个数组各自往前探 k//2 的距离
            new_index1 = min(index1 + k // 2 - 1, len(nums1) - 1)
            new_index2 = min(index2 + k // 2 - 1, len(nums2) - 1)
            
            pivot1, pivot2 = nums1[new_index1], nums2[new_index2]
            
            # 谁小，谁的前截就可以全部扔掉不看
            if pivot1 <= pivot2:
                # 扔掉了这么多个元素，k 就对应减少
                k -= new_index1 - index1 + 1
                index1 = new_index1 + 1
            else:
                k -= new_index2 - index2 + 1
                index2 = new_index2 + 1

    total_length = len(nums1) + len(nums2)
    # 不论奇数偶数，分别用除法计算寻找目标，偶数就是两个中心值取个平均
    if total_length % 2 == 1:
        return getKthElement((total_length + 1) // 2)
    else:
        return (getKthElement(total_length // 2) + getKthElement(total_length // 2 + 1)) / 2.0

### 搜索旋转排序数组（难度：中等）

**题目：**
整数数组 nums 按升序排列，数组中的值互不相同。在传递给函数之前，nums 在预先未知的某个下标 k 上进行了旋转。给你旋转后的数组 nums 和一个整数 target，如果 nums 中存在这个目标值 target，则返回它的下标，否则返回 -1。

**答案：**
```python
def search(nums, target):
    left, right = 0, len(nums) - 1
    while left <= right:
        mid = (left + right) // 2
        if nums[mid] == target:
            return mid
        # 左半部分有序
        if nums[left] <= nums[mid]:
            if nums[left] <= target < nums[mid]:
                right = mid - 1
            else:
                left = mid + 1
        # 右半部分有序
        else:
            if nums[mid] < target <= nums[right]:
                left = mid + 1
            else:
                right = mid - 1
    return -1
```

### 寻找两个正序数组的中位数（难度：困难）

**题目：**
给定两个大小分别为 m 和 n 的正序（从小到大）数组 nums1 和 nums2。请你找出并返回这两个正序数组的中位数。

**答案：**
```python
# 二分查找，时间复杂度 O(log(m+n))
def findMedianSortedArrays(nums1, nums2):
    if len(nums1) > len(nums2):
        nums1, nums2 = nums2, nums1
    
    m, n = len(nums1), len(nums2)
    left, right = 0, m
    
    while left <= right:
        partition1 = (left + right) // 2
        partition2 = (m + n + 1) // 2 - partition1
        
        maxLeft1 = float('-inf') if partition1 == 0 else nums1[partition1 - 1]
        minRight1 = float('inf') if partition1 == m else nums1[partition1]
        maxLeft2 = float('-inf') if partition2 == 0 else nums2[partition2 - 1]
        minRight2 = float('inf') if partition2 == n else nums2[partition2]
        
        if maxLeft1 <= minRight2 and maxLeft2 <= minRight1:
            if (m + n) % 2 == 0:
                return (max(maxLeft1, maxLeft2) + min(minRight1, minRight2)) / 2
            else:
                return max(maxLeft1, maxLeft2)
        elif maxLeft1 > minRight2:
            right = partition1 - 1
        else:
            left = partition1 + 1
```

### 格雷编码（难度：中等）

**题目：**
n 位格雷码序列是一个由 2^n 个整数组成的序列，其中每个整数都在范围 [0, 2^n - 1] 内，第一个整数是 0，每个整数在序列中出现最多一次，每对相邻整数值的二进制表示恰好一位不同，第一个和最后一个整数的二进制表示恰好一位不同。给你一个整数 n，返回任一有效的 n 位格雷码序列。

**答案：**
```python
def grayCode(n):
    res = [0]
    for i in range(n):
        for j in range(len(res) - 1, -1, -1):
            res.append(res[j] | (1 << i))
    return res
```
**复杂度：** 时间 O(2^n)，空间 O(1)

**关键点：** 对称法构造，每次在末尾添加镜像
