## 一、动态规划核心思想与打表套路

动态规划（DP）本质是一种**利用历史记录，避免重复计算**的优化思想。面试中百分之九十的 DP 题，只要能写出**状态转移方程**，代码就迎刃而解。

**核心做题骨架（DP五部曲）**：
1. **定义 dp 数组的物理含义**：`dp[i]` 到底代表什么？是“到第 i 阶楼梯的方法数”还是“以 nums[i] 结尾的最长子序列”？
2. **写出状态转移方程**：今天的状态由昨天哪些状态推导而来？（如 `dp[i] = dp[i-1] + dp[i-2]`）
3. **初始化基点**：起跑线在哪里？（如 `dp[0] = 0, dp[1] = 1`）
4. **确定遍历顺序**：是从左到右，还是从右到左？对于背包问题，内外层循环顺序极为讲究！
5. **举例打印 dp 数组验证**。

## 二、基础一维序列 DP

### 爬楼梯（难度：简单）

**题目：**
假设你正在爬楼梯。需要 `n` 阶你才能到达楼顶。每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？

**【模板解析与思路归纳】：**
- `dp[i]` 表示爬到第 `i` 阶楼梯的方法数。
- 最后一阶 `i` 要么是从 `i-1` 跨 1 步上来的，要么是从 `i-2` 跨 2 步上来的。所以 `dp[i] = dp[i-1] + dp[i-2]`（这就是彻底的斐波那契数列）。
- 空间优化：既然第 `i` 阶只和前面两阶有关，我们完全不需要建一个数组，只用两个变量滚动前进即可。

**答案代码（Python版 - 空间优化法）：**
```python
class Solution:
    def climbStairs(self, n: int) -> int:
        if n <= 2:
            return n
            
        # dp1 代表 dp[i-2], dp2 代表 dp[i-1]
        dp1, dp2 = 1, 2
        for _ in range(3, n + 1):
            dp1, dp2 = dp2, dp1 + dp2
            
        return dp2
```

### 打家劫舍（难度：中等）

**题目：**
小偷计划偷窃沿街的房屋，相邻的房屋装有互连的防盗系统（如果两间相邻的房屋在同一晚上被小偷闯入，系统会自动报警）。给定一个代表每个房屋存放金额的非负整数数组，计算在不触动警报的情况下，一夜之内能够偷窃到的最高金额。

**【模板解析与思路归纳】：**
- `dp[i]` 代表考虑前 `i` 间房子，所能偷到的最高金额。
- 面对第 `i` 间房，小偷只有两个选择：
  1. **偷这间房**：那么第 `i-1` 间房绝不能偷。此时金额为：`dp[i-2] + nums[i]`
  2. **不偷这间房**：那么你可以安心带走前 `i-1` 间房偷来的最高金额：`dp[i-1]`
- 状态转移方程：`dp[i] = max(dp[i-2] + nums[i], dp[i-1])`
- 同样可以采用双变量滚动优化空间。

**答案代码（Python版 - 空间优化法）：**
```python
class Solution:
    def rob(self, nums: List[int]) -> int:
        # prev2 = dp[i-2], prev1 = dp[i-1]
        prev2, prev1 = 0, 0
        
        for num in nums:
            prev2, prev1 = prev1, max(prev2 + num, prev1)
            
        return prev1
```

## 三、一维状态挂载（以自身结尾）

### 最长递增子序列 (LIS)（难度：中等）

**题目：**
给你一个整数数组 `nums`，找到其中最长严格递增子序列的长度。

**【模板解析与思路归纳】：**
- `dp[i]` 的定义并非“前 i 个元素中最长子序列的长度”，而是“**强制以 nums[i] 结尾的**最长递增子序列的长度”。这是序列 DP 最经典的强制挂载定义法。
- 转移方程：让 `j` 从 `0` 遍历到 `i-1`。只要发现 `nums[i] > nums[j]`，说明 `nums[i]` 可以接在 `nums[j]` 的屁股后面形成一个更长的递增序列！则 `dp[i] = max(dp[i], dp[j] + 1)`。

**答案代码（Python版 - DP O(N^2)）：**
```python
class Solution:
    def lengthOfLIS(self, nums: List[int]) -> int:
        if not nums:
            return 0
            
        # 所有元素各自本身就是个长度为 1 的子序列
        dp = [1] * len(nums)
        
        for i in range(len(nums)):
            for j in range(i):
                if nums[i] > nums[j]:
                    dp[i] = max(dp[i], dp[j] + 1)
                    
        return max(dp)
```
*(注：本题还存在 `贪心 + 二分查找` 达成 `O(N log N)` 的高级解法，建议在二分查找篇复习)*

## 四、背包问题 (Knapsack)

### 零钱兑换（难度：中等）

**题目：**
给你一个整数数组 `coins`，表示不同面额的硬币；以及一个整数 `amount`，表示总金额。计算并返回可以凑成总金额所需的 **最少的硬币个数**。如果没法凑出，返回 `-1`。可以认为每种硬币的数量是无限的。

**【模板解析与思路归纳】：**
- 这是经典的**完全背包求极值**问题。
- `dp[j]` 表示凑成金额 `j` 所需的最小硬币数。
- 既然是求最小，把所有 `dp` 初始化为正无穷 `float('inf')`，当然起跑线 `dp[0] = 0`（凑足 0 元需要 0 个硬币）。
- 转移方程：遍历面值 `coin`，如果当前目标金额 `i >= coin`，则 `dp[i] = min(dp[i], dp[i - coin] + 1)`。

**答案代码（Python版）：**
```python
class Solution:
    def coinChange(self, coins: List[int], amount: int) -> int:
        # dp 数组初始化为无穷大，表示初期认为凑不出
        dp = [float('inf')] * (amount + 1)
        dp[0] = 0
        
        # 外层遍金额，内层遍历硬币（或者反过来也可以，因为求的是全组合的极值而不是排列数）
        for i in range(1, amount + 1):
            for coin in coins:
                if i >= coin:
                    dp[i] = min(dp[i], dp[i - coin] + 1)
                    
        return dp[amount] if dp[amount] != float('inf') else -1
```

### 单词拆分（难度：中等）

**题目：**
给你一个字符串 `s` 和一个字符串列表 `wordDict` 作为字典。请你判断是否可以利用字典中出现的单词拼接出 `s`。注意：不要求字典中出现的单词全部都使用，并且字典中的单词可以重复使用。

**【模板解析与思路归纳】：**
- 这也是**完全背包问题**的变种，单词就是物品，字符串 `s` 就是背包！不同的是，这题要求的是**排列顺序必须严格一致**。
- `dp[i]` 代表字符串前 `i` 个字符组成的子串（即 `s[0:i]`）能否被字典里的单词拼出来。
- 转移：我们可以用 `j` 这个切入点将 `s[0:i]` 劈成两半：`s[0:j]` （也就是 `dp[j]`）和后缀 `s[j:i]`。如果 `dp[j]` 为 `True`，且后缀刚好在字典里，那拼合成功！`dp[i] = True`。

**答案代码（Python版 - 切分挂接法）：**
```python
class Solution:
    def wordBreak(self, s: str, wordDict: List[str]) -> bool:
        word_set = set(wordDict)  # 换成哈希集合，提速百倍
        n = len(s)
        
        # dp[i] 代表 s 的前 i 个字符组成的子串，能否被合法拼出
        dp = [False] * (n + 1)
        dp[0] = True  # 空串默认是可以拼出的（什么都不用选）
        
        for i in range(1, n + 1):
            # 将前 i 个字符的串一分为二，找一个能接住的断点 j
            for j in range(i):
                if dp[j] and s[j:i] in word_set:
                    dp[i] = True
                    break  # 只要有一种切法能成功就够了！
                    
        return dp[n]
```
