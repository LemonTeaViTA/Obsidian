## 一、动态规划核心思想与解题模板

动态规划（DP）的核心是为了**避免重复计算**。只要问题存在“重叠子问题”并且可以通过前面的最优状态推导出当前的最优状态（最优子结构），就可以套用DP分析。

### 🌟 三步走通用解题模板
无论题目多难，第一步必须强迫自己想清楚：
1. **明确定义 `dp` 数组的下标和含义**： `dp[i]`（一维）或 `dp[i][j]`（二维）到底代表着什么值？
2. **找出状态转移的递推公式（推断过场）**： 当前的 `dp[i]` 可以由之前的哪些状态计算得出？（最考思维逻辑）
3. **确定边界条件与最简单的状态初始化**： `dp[0]` 和 `dp[1]` 这种无法逆推的端点初始是多少？

*(注：以下题目按照从易入深的底层模型归类重整，请体会模型递进和抽象的过程。)*

## 二、一维基础状态转移
*这是最基础的DP。当前状态仅仅取决于前面直接相邻的状态（通常是它的前一项或前两项）。*

### 爬楼梯（难度：简单）

**题目：**
假设你正在爬楼梯。需要 n 阶你才能到达楼顶。每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？

**【模板解析与思路归纳】：**
- **1. 定义**：`dp[i]` 为爬到第 `i` 阶楼梯的方法总数。
- **2. 转移方程**：到达第 `i` 阶要么跨1步、要么跨2步来的。即 `dp[i] = dp[i-1] + dp[i-2]`。
- **3. 初始化**：最基础 `dp[1]=1, dp[2]=2`。*(注：代码中运用“滚动变量”优化技巧，仅用2个变量存储历史状态，极度省空间)*

**答案代码：**
```python
```python
# 动态规划，斐波那契数列
def climbStairs(n):
    if n <= 2:
        return n
    dp1, dp2 = 1, 2
    for _ in range(3, n + 1):
        dp1, dp2 = dp2, dp1 + dp2
    return dp2
```

### 打家劫舍（难度：中等）

**题目：**
你是一个专业的小偷，计划偷窃沿街的房屋。每间房内都藏有一定的现金，影响你偷窃的唯一制约因素就是相邻的房屋装有相互连通的防盗系统，如果两间相邻的房屋在同一晚上被小偷闯入，系统会自动报警。给定一个代表每个房屋存放金额的非负整数数组，计算你不触动警报装置的情况下，一夜之内能够偷窃到的最高金额。

**【模板解析与思路归纳】：**
- **1. 定义**：`dp[i]` 表示考虑前 `i` 间房子能偷到的最高金额。
- **2. 转移方程**：对于当前房子，不偷，收益是 `dp[i-1]`；偷，必定不能偷前一家，加上前面的最高即 `dp[i-2] + nums[i]`。取两者最大：`dp[i] = max(dp[i-1], dp[i-2] + nums[i])`。

**答案代码：**
```python
```python
def rob(nums):
    if not nums:
        return 0
    if len(nums) == 1:
        return nums[0]
    dp1, dp2 = 0, 0
    for num in nums:
        dp1, dp2 = dp2, max(dp1 + num, dp2)
    return dp2
```

### 最大子数组和（难度：中等）

**题目：**
给你一个整数数组 nums，请你找出一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。

**【模板解析与思路归纳】：**
- **1. 定义**：`dp[i]` 表示以第 `i` 个数字**必定结尾**的连续子数组的最大和。
- **2. 转移递推**：如果之前的和变成了负数（拖累），加进去反而变小，不如从自己算起；如前面为正数，加自己会变更大。所以 `dp[i] = max(nums[i], dp[i-1] + nums[i])`。

**答案代码：**
```python
```python
# 动态规划（Kadane 算法）
def maxSubArray(nums):
    max_sum = nums[0]
    current_sum = nums[0]
    for i in range(1, len(nums)):
        current_sum = max(nums[i], current_sum + nums[i])
        max_sum = max(max_sum, current_sum)
    return max_sum
```

### 比特位计数（难度：简单）

**题目：**
给你一个整数 n，对于 0 <= i <= n 中的每个 i，计算其二进制表示中 1 的个数，返回一个长度为 n + 1 的数组 ans 作为答案。

**【模板解析与思路归纳】：**
- 位运算套用DP。
- 偶数的1的数量和其一半除以2的数字相同；奇数比减去1的偶数多一个1（即最右位变成1）。`dp[i] = dp[i >> 1] + (i & 1)`。

**答案代码：**
```python
```python
def countBits(n):
    dp = [0] * (n + 1)
    for i in range(1, n + 1):
        dp[i] = dp[i >> 1] + (i & 1)
    return dp
```

## 三、经典双变量/内外循环套路（背包问题）
*“背包九讲”的入门，往往是双层循环嵌套求解。这类题一定要区分两者的遍历顺序区别！*
* **0-1背包**（数字只能选一次）：内层遍历必须**从大到小（倒序被动）**，防同数字重用。
* **完全背包**（数字无限制选）：内层遍历必须**从小到大（正序贪心）**以容许重复。

### 分割等和子集（难度：中等）

**题目：**
给你一个只包含正整数的非空数组 nums。请你判断是否可以将这个数组分割成两个子集，使得两个子集的元素和相等。

**【模板解析与思路归纳】：**
- **模板匹配**：总和必须正好切成两半，等价为**“用容量为 sum/2 的 0-1背包”**能不能恰好塞满。
- **定义**：`dp[j]` 值为 `j` 的容量是否可被组成。
- **方程**：要么不选当前的数字，要么选（腾出容量来继承前置布尔值） `dp[j] = dp[j] or dp[j - nums[i]]`。
- **细节**：0-1背包（数字只准一用），所以**内层包容量 `j` 倒序遍历**！

**答案代码：**
```python
```python
def canPartition(nums):
    total = sum(nums)
    if total % 2 != 0:
        return False
    target = total // 2
    dp = [False] * (target + 1)
    dp[0] = True
    for num in nums:
        for j in range(target, num - 1, -1):
            dp[j] = dp[j] or dp[j - num]
    return dp[target]
```

### 目标和（难度：中等）

**题目：**
给你一个非负整数数组 nums 和一个整数 target。向数组中的每个整数前添加 '+' 或 '-'，然后串联起所有整数，构造一个表达式。返回可以通过上述方法构造的、运算结果等于 target 的不同表达式的数目。

**【模板解析与思路归纳】：**
- **模板变形**：数学变换：因为 `P - N = tar`, `P + N = sum`, 加起来得出正数和必须挑满：`P = (tar + sum) / 2`。这题**也回到了填满 0-1背包的方法数**模型上。
- **方程**：`dp[j] += dp[j - num]`。依然是倒序容量。

**答案代码：**
```python
```python
def findTargetSumWays(nums, target):
    total = sum(nums)
    if abs(target) > total or (total + target) % 2 != 0:
        return 0
    positive = (total + target) // 2
    dp = [0] * (positive + 1)
    dp[0] = 1
    for num in nums:
        for j in range(positive, num - 1, -1):
            dp[j] += dp[j - num]
    return dp[positive]
```

### 零钱兑换（难度：中等）

**题目：**
给你一个整数数组 coins，表示不同面额的硬币；以及一个整数 amount，表示总金额。计算并返回可以凑成总金额所需的最少的硬币个数。如果没有任何一种硬币组合能组成总金额，返回 -1。

**【模板解析与思路归纳】：**
- **模板匹配**：因为金币数量无限个，所以这是标准的**完全背包**问题（求极小值）！
- **定义**：`dp[j]` 为刚好凑成 `j` 钱数最少需要的硬币枚数。
- **方程**：`dp[j] = min(dp[j], dp[j - coin] + 1)`。
- **细节**：因为硬币可以反复制用，所以**内层遍历 `j` 正序**覆盖掉以前的最优解！

**答案代码：**
```python
```python
def coinChange(coins, amount):
    dp = [float('inf')] * (amount + 1)
    dp[0] = 0
    for coin in coins:
        for i in range(coin, amount + 1):
            dp[i] = min(dp[i], dp[i - coin] + 1)
    return dp[amount] if dp[amount] != float('inf') else -1
```

## 四、二维网格平面DP (从左上打到右下)
*在这类迷宫或地图问题中，状态与平面位置(i,j)绑定。递推方程往往看你的步向是从左来还是上方来。*

### 不同路径（难度：中等）

**题目：**
一个机器人位于一个 m x n 网格的左上角。机器人每次只能向下或者向右移动一步。机器人试图达到网格的右下角。问总共有多少条不同的路径？

**【模板解析与思路归纳】：**
- **1. 定义**：`dp[i][j]`为抵达网格点 `(i, j)` 累积的方法总数。
- **2. 转移递推**：机器人只能往左或者下走；如果站在终点往回想，自己肯定是从 上方 或者 左侧 的格子走过来的即 `dp[i][j] = dp[i-1][j] + dp[i][j-1]`。

**答案代码：**
```python
```python
# 动态规划
def uniquePaths(m, n):
    dp = [[1] * n for _ in range(m)]
    for i in range(1, m):
        for j in range(1, n):
            dp[i][j] = dp[i-1][j] + dp[i][j-1]
    return dp[m-1][n-1]

# 数学组合 C(m+n-2, m-1)
def uniquePaths(m, n):
    import math
    return math.comb(m + n - 2, m - 1)
```

### 最小路径和（难度：中等）

**题目：**
给定一个包含非负整数的 m x n 网格 grid，请找出一条从左上角到右下角的路径，使得路径上的数字总和为最小。每次只能向下或者向右移动一步。

**【模板解析与思路归纳】：**
- 与《不同路径》如出一辙，目标换成了求局部最小的和而已。
- `dp[i][j] = min(dp[i-1][j], dp[i][j-1]) + grid[i][j]`。

**答案代码：**
```python
```python
def minPathSum(grid):
    m, n = len(grid), len(grid[0])
    dp = [[0] * n for _ in range(m)]
    dp[0][0] = grid[0][0]
    for i in range(1, m):
        dp[i][0] = dp[i-1][0] + grid[i][0]
    for j in range(1, n):
        dp[0][j] = dp[0][j-1] + grid[0][j]
    for i in range(1, m):
        for j in range(1, n):
            dp[i][j] = min(dp[i-1][j], dp[i][j-1]) + grid[i][j]
    return dp[m-1][n-1]
```

### 最大正方形（难度：中等）

**题目：**
在一个由 '0' 和 '1' 组成的二维矩阵内，找到只包含 '1' 的最大正方形，并返回其面积。

**【模板解析与思路归纳】：**
- 网格的升级形态。
- 构成大正方形的关键：以自己 `(i,j)` 为右下角的角，必须上面、左边、左上全都是合法的 1。这三个边长哪怕有一个有短板（出现0），都没法膨胀。也就是“取木桶最短板”的原理：`dp[i][j] = min(dp[i-1][j], dp[i][j-1], dp[i-1][j-1]) + 1`。

**答案代码：**
```python
```python
def maximalSquare(matrix):
    if not matrix:
        return 0
    m, n = len(matrix), len(matrix[0])
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    max_side = 0
    
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if matrix[i-1][j-1] == '1':
                dp[i][j] = min(dp[i-1][j], dp[i][j-1], dp[i-1][j-1]) + 1
                max_side = max(max_side, dp[i][j])
    
    return max_side * max_side
```

## 五、打擂台子序列与组合匹配DP
*这类问题要求挑出一个符合规则的“序列”。子序列问题需要不断用 `num[i]` 回头去和全长度前面所有的 `num[j]` 作比较并合并最长值。*

### 最长递增子序列（难度：中等）

**题目：**
给你一个整数数组 nums，找到其中最长严格递增子序列的长度。

**【模板解析与思路归纳】：**
- **这道题开局非常考“定义”**。
- **1. 定义**：`dp[i]`为必须以当前项 `nums[i]` **结尾**的LIS极长长度。
- **2. 双层循环与转移**：当前数既然要递增，那就向前挨个翻烂账 `j < i`，如果碰见 `nums[j] < nums[i]`，意味着 `i` 可以完美接轨，跟在 `j` 组成的子序列尾部，长度就顺理成章 `dp[i] = max(dp[i], dp[j] + 1)`！

**答案代码：**
```python
```python
# 动态规划 O(n²)
def lengthOfLIS(nums):
    if not nums:
        return 0
    dp = [1] * len(nums)
    for i in range(len(nums)):
        for j in range(i):
            if nums[j] < nums[i]:
                dp[i] = max(dp[i], dp[j] + 1)
    return max(dp)

# 贪心 + 二分 O(n log n)
def lengthOfLIS(nums):
    tails = []
    for num in nums:
        left, right = 0, len(tails)
        while left < right:
            mid = (left + right) // 2
            if tails[mid] < num:
                left = mid + 1
            else:
                right = mid
        if left == len(tails):
            tails.append(num)
        else:
            tails[left] = num
    return len(tails)
```

### 最长回文子序列（难度：中等）

**题目：**
给你一个字符串 s，找出其中最长的回文子序列，并返回该序列的长度。子序列定义为：不改变剩余字符顺序的情况下，删除某些字符或者不删除任何字符形成的一个序列。

**【模板解析与思路归纳】：**
- 回文要两头查！典型的**区间DP起手**。
- `dp[i][j]` 定义为字符串 `s` 切取区间 `[i, j]` 中的内含最长回文长度。
- 转移：两个头相等 `s[i] == s[j]`，内部夹断的就是 `dp[i+1][j-1]` 长2。
- 若不等，说明至少某个端点的字符配不成对，踢掉一边，取最大剩余值 `max(dp[i+1][j], dp[i][j-1])`。

**答案代码：**
```python
```python
def longestPalindromeSubseq(s):
    n = len(s)
    dp = [[0] * n for _ in range(n)]
    
    for i in range(n - 1, -1, -1):
        dp[i][i] = 1
        for j in range(i + 1, n):
            if s[i] == s[j]:
                dp[i][j] = dp[i+1][j-1] + 2
            else:
                dp[i][j] = max(dp[i+1][j], dp[i][j-1])
    
    return dp[0][n-1]
```

### 单词拆分（难度：中等）

**题目：**
给你一个字符串 s 和一个字符串列表 wordDict 作为字典。请你判断是否可以利用字典中出现的单词拼接出 s。

**【模板解析与思路归纳】：**
- **1. 定义**：`dp[i]` 代表句子切取前 `i` 长度个字符正好能全部让字典解释完对不对（布尔值True/False）。
- **2. 循环切取**：从左遍历，在 `i` 里截出内部截点 `j`，一旦它自己的前半部分拼得出，并且剩余的 `s[j:i]` 是个好单词，全段就算成功啦。

**答案代码：**
```python
```python
def wordBreak(s, wordDict):
    wordSet = set(wordDict)
    dp = [False] * (len(s) + 1)
    dp[0] = True
    for i in range(1, len(s) + 1):
        for j in range(i):
            if dp[j] and s[j:i] in wordSet:
                dp[i] = True
                break
    return dp[len(s)]
```

### 完全平方数（难度：中等）

**题目：**
给你一个整数 n，返回和为 n 的完全平方数的最少数量。完全平方数是一个整数，其值等于另一个整数的平方。

**【模板解析与思路归纳】：**
- 把“所有的完全平方数”当成数量无限的铜板，它就是在找最少找了几枚完全硬币凑满，跟前面的《零钱兑换》题一毛一样的 **完全背包套路**。

**答案代码：**
```python
```python
def numSquares(n):
    dp = [float('inf')] * (n + 1)
    dp[0] = 0
    for i in range(1, n + 1):
        j = 1
        while j * j <= i:
            dp[i] = min(dp[i], dp[i - j*j] + 1)
            j += 1
    return dp[n]
```

### 解码方法（难度：中等）

**题目：**
一条包含字母 A-Z 的消息通过以下映射进行了编码：'A' -> "1", 'B' -> "2", ..., 'Z' -> "26"。要解码已编码的消息，所有数字必须基于上述映射的方法，反向映射回字母。请你计算并返回解码方法的总数。

**【模板解析与思路归纳】：**
- 这不就是加了限制变种的《爬楼梯》？
- 每一格既可以选择单独自己（1位变成单词），前提不是 '0'；也能够牵手前一位数字（2个合并），前提加起来的值限定 `<= 26`。分类加进去罢了。

**答案代码：**
```python
```python
def numDecodings(s):
    if not s or s[0] == '0':
        return 0
    dp = [0] * (len(s) + 1)
    dp[0] = dp[1] = 1
    for i in range(2, len(s) + 1):
        if s[i-1] != '0':
            dp[i] += dp[i-1]
        if 10 <= int(s[i-2:i]) <= 26:
            dp[i] += dp[i-2]
    return dp[len(s)]

## 六、高级困难（二维字符串状态与区间爆炸）
*这里是笔试压轴防拿高分的噩梦。DP矩阵开始变成在两副字符串之间作对比，或者打碎重组成复杂的三角区间。*

### 最长公共子序列（难度：中等）

**题目：**
给定两个字符串 text1 和 text2，返回这两个字符串的最长公共子序列的长度。如果不存在公共子序列，返回 0。

**【模板解析与思路归纳】：**
- **1. 定义矩阵**：经典的二维字符串对比 DP。`dp[i][j]` 表示串 A 的 `0..i` 段和串 B 的 `0..j` 段共有几个字母匹配。
- **2. 抛离规律方程**：
    如果结尾字符碰巧碰上了 `A[i]==B[j]`，好极了双双牵手：`dp[i-1][j-1] + 1`；
    哪怕有一个不相等，那就必然有人“多余不进组合”，我们试着抛弃各自一边比较取大值 `max(dp[i-1][j], dp[i][j-1])`。

**答案代码：**
```python
```python
def longestCommonSubsequence(text1, text2):
    m, n = len(text1), len(text2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if text1[i-1] == text2[j-1]:
                dp[i][j] = dp[i-1][j-1] + 1
            else:
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])
    return dp[m][n]

### 编辑距离（难度：困难）

**题目：**
给你两个单词 word1 和 word2，请你计算出将 word1 转换成 word2 所使用的最少操作数。你可以对一个单词进行如下三种操作：插入一个字符、删除一个字符、替换一个字符。

**【模板解析与思路归纳】：**
- **这是动态规划界赫赫有名的《增删改模板大BOSS》**。
- `dp[i][j]` 的定义变成字符串互改的最少步数。
- 你要想清楚：`dp[i-1][j] + 1` 意味着**单向增加/删除一次。**
- `dp[i][j-1] + 1` 意味着**增加/删除操作。**
- `dp[i-1][j-1] + 1` 意味着**如果不等进行原地替换字操作**。
- 所有状态互相比对包圆这几套小动作。

**答案代码：**
```python
```python
def minDistance(word1, word2):
    m, n = len(word1), len(word2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    
    for i in range(m + 1):
        dp[i][0] = i
    for j in range(n + 1):
        dp[0][j] = j
    
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if word1[i-1] == word2[j-1]:
                dp[i][j] = dp[i-1][j-1]
            else:
                dp[i][j] = min(dp[i-1][j], dp[i][j-1], dp[i-1][j-1]) + 1
    
    return dp[m][n]

### 最长有效括号（难度：困难）

**题目：**
给你一个只包含 '(' 和 ')' 的字符串，找出最长有效（格式正确且连续）括号子串的长度。

**【模板解析与思路归纳】：**
- 这里需要用的一维套路简直心机。当当前匹配到了 `)`，且有前面的 `(` 可做接应时，中间可能本身就藏有很长的一个正确的大括弧，得跨过那个长的部分接在尾巴前段。这个坐标索引边界很易翻车。

**答案代码：**
```python
```python
# 动态规划
def longestValidParentheses(s):
    if not s:
        return 0
    dp = [0] * len(s)
    max_len = 0
    for i in range(1, len(s)):
        if s[i] == ')':
            if s[i-1] == '(':
                dp[i] = (dp[i-2] if i >= 2 else 0) + 2
            elif i - dp[i-1] > 0 and s[i - dp[i-1] - 1] == '(':
                dp[i] = dp[i-1] + (dp[i - dp[i-1] - 2] if i - dp[i-1] >= 2 else 0) + 2
        max_len = max(max_len, dp[i])
    return max_len
```

### 戳气球（难度：困难）

**题目：**
有 n 个气球，编号为 0 到 n - 1，每个气球上都标有一个数字，这些数字存在数组 nums 中。现在要求你戳破所有的气球。戳破第 i 个气球，你可以获得 nums[i - 1] * nums[i] * nums[i + 1] 枚硬币。这里的 i - 1 和 i + 1 代表和 i 相邻的两个气球的序号。如果 i - 1 或 i + 1 超出了数组的边界，那么就当它是一个数字为 1 的气球。求所能获得硬币的最大数量。

**【模板解析与思路归纳】：**
- 它的脑洞实在难以突破，被称为**区间DP的死循环终点站**。
- 直接去想“下一个戳那个”会把数组戳得七零八落，失去规律重组；**妙招解构：反转脑回路逆向思考**，想想假设 `k` 就是**最后一个引爆的气球**！它不爆之前，左边归左边，右半区归右边，它们再怎么戳也不影响核心区！这才推出完美干净的 `dp[i][j] = dp[i][k] + dp[k][j] + (nums[i]*nums[k]*nums[j])` 的三角闭环。

**答案代码：**
```python
```python
def maxCoins(nums):
    nums = [1] + nums + [1]
    n = len(nums)
    dp = [[0] * n for _ in range(n)]
    
    for length in range(2, n):
        for left in range(n - length):
            right = left + length
            for k in range(left + 1, right):
                coins = dp[left][k] + dp[k][right] + nums[left] * nums[k] * nums[right]
                dp[left][right] = max(dp[left][right], coins)
    
    return dp[0][n-1]
```
