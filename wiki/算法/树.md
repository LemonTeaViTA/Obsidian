## 一、二叉树核心思想与终极模板

二叉树题目的本质只有两种解法思路：**“遍历一遍树（DFS/BFS）”** 或者 **“分解问题（分治/递归）”**。
绝大多数难解的树的题目，靠的都是**后序遍历（Bottom-Up 分治）**：即先向左右子树要答案，结合当前节点的情况，再把答案往上返回给父节点。

### 🌟 递归的“退元法”三大灵魂拷问
写递归时永远不要人脑压栈，只要想清楚：
1. **这个函数是干嘛的？**（明确定义和返回值）
2. **这个函数参数中的节点要做什么？**（当前节点的单层逻辑是什么，什么时候做操作：前、中、后序）
3. **递归的出口（base case）是什么？**（通常是 `if not root: return ...`）

## 二、基础入门：树的遍历 (Traversal)
*掌握递归与迭代形式的树的遍历是所有树形 DP 的基础。*

### 二叉树的中序遍历（难度：简单）

**题目：**
给定一个二叉树的根节点 `root`，返回它的中序遍历（左 -> 根 -> 右）。

**【模板解析与思路归纳】：**
- 中序遍历非常适合二叉搜索树（BST），因为 BST 的中序遍历就是一个**递增**序列。
- 递归写法极其简单。迭代写法需要借助栈（模拟系统的函数调用栈），一路向左冲到底。

**答案代码（Python版）：**
```python
class Solution:
    # --- 解法 1: 递归法 ---
    def inorderTraversal_recursive(self, root: TreeNode) -> List[int]:
        res = []
        def dfs(node):
            if not node:
                return
            dfs(node.left)
            res.append(node.val)
            dfs(node.right)
        dfs(root)
        return res
        
    # --- 解法 2: 迭代法（常用模板） ---
    def inorderTraversal(self, root: TreeNode) -> List[int]:
        res, stack = [], []
        curr = root
        
        while curr or stack:
            # 1. 一直向左走，把沿途节点全压入栈
            while curr:
                stack.append(curr)
                curr = curr.left
            # 2. 走到头了，弹出一个节点进行处理 (这是中序遍历最左侧的那个)
            curr = stack.pop()
            res.append(curr.val)
            # 3. 转向该节点的右子树
            curr = curr.right
            
        return res
```

### 二叉树的层序遍历（难度：中等）

**题目：**
给你二叉树的根节点 `root` ，返回其节点值的层序遍历。 （即逐层地，从左到右访问所有节点）。

**【模板解析与思路归纳】：**
- 纯粹的 **BFS（广度优先搜索）**，必须借助队列（Queue）。
- 为了实现“按层分组”（返回 `List[List[int]]`），每次 `while queue:` 开始时，**首先记录当前队列的长度 `size = len(queue)`**，然后进行 `for _ in range(size)` 循环。这就保证了一次 for 循环恰好把“当前这一层”全部处理完！

**答案代码（Python版）：**
```python
from collections import deque

class Solution:
    def levelOrder(self, root: TreeNode) -> List[List[int]]:
        if not root:
            return []
            
        res = []
        queue = deque([root])
        
        while queue:
            level_nodes = []
            size = len(queue)  # 核心：记录当前层的节点数
            
            for _ in range(size):
                node = queue.popleft()
                level_nodes.append(node.val)
                # 将下一层的孩子加入队列
                if node.left:
                    queue.append(node.left)
                if node.right:
                    queue.append(node.right)
                    
            res.append(level_nodes)
            
        return res
```

## 三、进阶：属性与结构变换 (Attributes & Structure)
*这类题目通常采用分解问题的思路。递归处理左半边和右半边，再结合根节点。*

### 二叉树的最大深度（难度：简单）

**【模板解析与思路归纳】：**
- **分治法思想**：整棵树的最大深度，等于 `左子树的最大深度` 和 `右子树的最大深度` 中的较大者，再加上 1（根节点自己）。这就是最经典的后序遍历（从下往上收集结果）。

**答案代码（Python版）：**
```python
class Solution:
    def maxDepth(self, root: TreeNode) -> int:
        if not root:
            return 0
        
        left_depth = self.maxDepth(root.left)
        right_depth = self.maxDepth(root.right)
        
        return 1 + max(left_depth, right_depth)
```

### 翻转二叉树（难度：简单）

**题目：**
给你一棵二叉树的根节点 `root` ，翻转这棵二叉树，并返回其根节点。*(注：著名的 Homebrew 作者因写不出这道题被拒).*

**【模板解析与思路归纳】：**
- 前序遍历/后序遍历皆可。对于当前节点，直接把它的左儿子和右儿子交换一下，然后让左儿子、右儿子各自去翻转它们自己的子树即可。

**答案代码（Python版）：**
```python
class Solution:
    def invertTree(self, root: TreeNode) -> TreeNode:
        if not root:
            return None
            
        # 交换左右子树
        root.left, root.right = root.right, root.left
        
        # 让左右子树继续去内部翻转
        self.invertTree(root.left)
        self.invertTree(root.right)
        
        return root
```

### 对称二叉树（难度：简单）

**题目：**
给你一个二叉树的根节点 `root` ， 检查它是否轴对称。

**【模板解析与思路归纳】：**
- 检查对称，实质上是要比较两棵树 `t1` 和 `t2`：
  1. 它们的值是否相等？
  2. `t1` 的左子树 和 `t2` 的右子树 是否对称？
  3. `t1` 的右子树 和 `t2` 的左子树 是否对称？

**答案代码（Python版）：**
```python
class Solution:
    def isSymmetric(self, root: TreeNode) -> bool:
        if not root:
            return True
            
        def check(left_node, right_node):
            if not left_node and not right_node:
                return True
            if not left_node or not right_node:
                return False
            if left_node.val != right_node.val:
                return False
                
            # 左边的左孩子 和 右边的右孩子 比较；左边的右孩子 和 右边的左孩子 比较
            return check(left_node.left, right_node.right) and check(left_node.right, right_node.left)
            
        return check(root.left, root.right)
```

## 四、高阶构造与深度搜索 (Advanced DFS)
*这部分开始考验对二叉树形态的掌控能力，通常需要更深度的后序遍历或者携带外部状态。*

### 二叉树的最近公共祖先 (LCA)（难度：中等）

**题目：**
给定一个二叉树, 找到该树中两个指定节点 p 和 q 的最近公共祖先。

**【模板解析与思路归纳】：**
- 这是非常非常经典的**后序递归**。我们在底下找到了 `p` 和 `q`，怎么把它向上传递？
- 对于当前节点 `root`：
  1. 如果 `root` 就是 `p` 或 `q`，直接返回 `root` 自爆身份（找到了！）。
  2. 去左边找 `left`，去右边找 `right`。
  3. 如果 `left` 和 `right` 都不为空，那说明 `p` `q` 一左一右，当前的 `root` 就是**最近公共祖先**！将其向上返回！
  4. 如果只找到一边，那就把找到的那一边继续往上抛。

**答案代码（Python版）：**
```python
class Solution:
    def lowestCommonAncestor(self, root: 'TreeNode', p: 'TreeNode', q: 'TreeNode') -> 'TreeNode':
        # Base case: 越界或者是直接碰到了p或q
        if not root or root == p or root == q:
            return root
            
        # 分治：去左右子树找
        left = self.lowestCommonAncestor(root.left, p, q)
        right = self.lowestCommonAncestor(root.right, p, q)
        
        # 如果左右两边都有收获，说明我(root)就是它们的结合点，LCA诞生！
        if left and right:
            return root
            
        # 如果只有一边有，那么把有的那一边继续向上托举
        return left if left else right
```

### 从前序与中序遍历序列构造二叉树（难度：中等）

**题目：**
给定两个整数数组 `preorder` 和 `inorder` ，其中 `preorder` 是先序（根->左->右），`inorder` 是中序（左->根->右），构造二叉树并返回其根节点。

**【模板解析与思路归纳】：**
- `preorder` 的第一个元素**永远是根节点**！
- 拿着这个根节点去 `inorder` 里面找，中序里根节点**左边的一堆就是左子树，右边的一堆就是右子树**。
- 得到左子树的长度后，也可以把 `preorder` 切成左右两半，递归建立即可。

**答案代码（Python版）：**
```python
class Solution:
    def buildTree(self, preorder: List[int], inorder: List[int]) -> TreeNode:
        if not preorder or not inorder:
            return None
            
        # 1. 抓出根节点
        root_val = preorder[0]
        root = TreeNode(root_val)
        
        # 2. 找到根节点在中序数组里的位置
        idx = inorder.index(root_val)
        
        # 3. 递归构造左右子树
        # idx 恰好也是左子树那堆数字的长度
        root.left = self.buildTree(preorder[1 : idx + 1], inorder[:idx])
        root.right = self.buildTree(preorder[idx + 1:], inorder[idx + 1:])
        
        return root
```

### 二叉树展开为链表（难度：中等）

**题目：**
给你二叉树的根结点 `root` ，请你将它展开为一个单链表，方向同样向右，左指针全部为 None。

**【模板解析与思路归纳】：**
- 通过寻找“前驱”的方式就地解决：其实每次你把左子树扒下来，塞到根节点和右子树中间即可。
- 需要将原本的右子树接到原本左子树的最右下角的节点上。

**答案代码（Python版）：**
```python
class Solution:
    def flatten(self, root: TreeNode) -> None:
        """
        Do not return anything, modify root in-place instead.
        """
        curr = root
        while curr:
            if curr.left:
                # 找左子树的最右端节点，那是这棵左子树在先序遍历里最后一个被访问的
                predecessor = curr.left
                while predecessor.right:
                    predecessor = predecessor.right
                    
                # 偷梁换柱：将原来的右子树接到这个左子树的最右角下
                predecessor.right = curr.right
                curr.right = curr.left
                curr.left = None  # 左边置空
                
            # 继续往下走
            curr = curr.right
```

## 五、特殊结构：前缀树 (Trie)

### 实现 Trie (前缀树)（难度：中等）

**题目：**
Trie 是一种树形数据结构，用于高效地存储和检索字符串数据集中的键。实现 `insert`, `search` 和 `startsWith` 方法。

**【模板解析与思路归纳】：**
- 字典树的本质：**节点不存完整的字符，而是依靠“边”指向下一个节点来代表字符的流转**。
- 用字典 `children` 可以极大地简化查找，再加一个布尔标记 `is_end` 代表是否有单词在此终结即可。

**答案代码（Python版）：**
```python
class Trie:
    def __init__(self):
        # 节点内不需要强行存储 char，它的存在由 parent 里面的 key 指向它来保证
        self.children = {}
        self.is_end = False

    def insert(self, word: str) -> None:
        node = self
        for char in word:
            if char not in node.children:
                node.children[char] = Trie()
            node = node.children[char]
        # 到达末尾，标记单词结束
        node.is_end = True

    def search(self, word: str) -> bool:
        node = self
        for char in word:
            if char not in node.children:
                return False
            node = node.children[char]
        return node.is_end

    def startsWith(self, prefix: str) -> bool:
        node = self
        for char in prefix:
            if char not in node.children:
                return False
            node = node.children[char]
        # 只要跑完了 prefix，不管有没有单词在这里结束，都算存在这个前缀
        return True
```
