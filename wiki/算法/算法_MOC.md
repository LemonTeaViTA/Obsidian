## 📖 14 大核心模块知识地图

按照数据结构与解题思想拆分。每个模块都有：**模式识别信号 → 终极模板 → 题目（思维推导链）**。

### 1. 基础数据结构与技巧

- 📦 **[[数组]]** — 双指针（对撞 / 快慢 / 同向）、前缀和、滑动窗口、哈希加速
- 🔤 **[[字符串]]** — 双指针、哈希计数、中心扩展、KMP 思维
- 🔗 **[[链表]]** — Dummy Head、快慢指针、穿针引线反转
- 📚 **[[栈与队列]]** — 单调栈、单调双端队列、括号匹配

### 2. 树与图的搜索

- 🌳 **[[树]]** — DFS / BFS、前中后序、公共祖先、BST、字典树
- 🌐 **[[图]]** — 网格搜索、拓扑排序、Dijkstra
- 🔄 **[[回溯算法]]** — 组合 / 排列 / 子集树模板与剪枝

### 3. 高级算法思想

- 💡 **[[动态规划]]** — 五部曲、序列 DP、挂载 DP、状态机 DP（主文档与思想总纲）
- 🎒 **[[动态规划-背包]]** — 0-1 背包 / 完全背包 / 多重背包 / 分组背包的模板与遍历顺序
- 🧱 **[[动态规划-多维]]** — 网格 DP / 区间 DP / 双字符串 DP（LCS / 编辑距离）
- 🎯 **[[贪心算法]]** — 不断刷新极限、局部最优推全局
- 🔍 **[[二分查找]]** — 三大模板、二分答案
- ⛰️ **[[堆]]** — Top K、多路归并、对顶堆找中位数

### 4. 杂项与底层

- 🧮 **[[位运算]]** — 异或全家桶、低位 1 / 高位 1 操作
- 📊 **[[排序]]** — 快排 / 归并手写、快速选择、区间合并、荷兰国旗
- ⚙️ **[[设计]]** — LRU、LFU 等容器设计

## 🛤️ 学习路径建议

以下是不同阶段的学习路径，**按面试权重和依赖关系排序**。

### Stage 1 · 入门必刷（建立思维框架）

> 目标：掌握最基础的数据结构操作，建立"问题 → 模式"的初步映射。

1. **[[数组]]** — 双指针起手式，几乎每场面试都会考
2. **[[链表]]** — 虚拟头 + 快慢指针的肌肉记忆
3. **[[栈与队列]]** — 理解 LIFO / FIFO，括号匹配为代表
4. **[[字符串]]** — 数组思维在字符上的延伸

### Stage 2 · 核心算法（面试重灾区）

> 目标：掌握"以套路解题"的核心思维。

5. **[[二分查找]]** — 三大模板必背，二分答案是高级题武器
6. **[[树]]** — 后序分治是解一切树形 DP 的钥匙
7. **[[回溯算法]]** — 模板固定，穷举所有可能
8. **[[贪心算法]]** — 局部最优解题，思维突破点

### Stage 3 · 算法分水岭

> 目标：能在 30 分钟内识别 DP 模式并写出状态转移方程。

9. **[[动态规划]]** — ==面试最大区分度==，必须建立"五部曲"思维（再深入 [[动态规划-背包]] / [[动态规划-多维]]）
10. **[[堆]]** — Top K 必杀技
11. **[[排序]]** — 快排 / 归并是面试官最爱的手撕题

### Stage 4 · 进阶专题

> 目标：处理图论和高频特殊技巧。

12. **[[图]]** — 网格 + 拓扑 + Dijkstra
13. **[[位运算]]** — 高频 trick，写出来面试官会眼前一亮
14. **[[设计]]** — LRU 是 BAT 必考
