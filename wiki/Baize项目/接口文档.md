# 接口文档

## 一、用户管理模块

### 1. 用户注册

- **URL**: `/api/v1/users/register`
- **Method**: POST
- **Request Body**:
```json
{
  "username": "string",   // 用户名，唯一
  "password": "string"    // 密码（明文传输，后端加密存储）
}
```
- **Response**:
```json
// 成功
{
  "code": 200,
  "message": "User registered successfully"
}

// 失败
{
  "code": 400,
  "message": "Username already exists"
}
```

### 2. 用户登录

- **URL**: `/api/v1/users/login`
- **Method**: POST
- **Request Body**:
```json
{
  "username": "string",
  "password": "string"
}
```
- **Response**:
```json
// 成功
{
  "code": 200,
  "message": "Login successful",
  "data": {
    "token": "JWT_TOKEN_STRING"
  }
}

// 失败
{
  "code": 401,
  "message": "Invalid username or password"
}
```

### 3. 获取用户信息

- **URL**: `/api/v1/users/me`
- **Method**: GET
- **Headers**: `Authorization: Bearer JWT_TOKEN_STRING`
- **Response**:
```json
{
  "code": 200,
  "data": {
    "id": 123,
    "username": "user1",
    "role": "USER",
    "orgTags": ["PRIVATE_user1", "TECH_DEPT"],
    "primaryOrg": "PRIVATE_user1",
    "createdAt": "2025-01-01T10:00:00Z"
  }
}
```

### 4. 更新用户信息

- **URL**: `/api/v1/users/{userId}`
- **Method**: PUT
- **Headers**: `Authorization: Bearer JWT_TOKEN_STRING`
- **Request Body**:
```json
{
  "orgTags": ["PRIVATE_user1", "TECH_DEPT", "BACKEND_TEAM"],
  "primaryOrg": "TECH_DEPT"
}
```
- **Response**:
```json
{
  "code": 200,
  "message": "User updated successfully"
}
```

### 5. 获取用户列表（管理员）

- **URL**: `/api/v1/admin/users`
- **Method**: GET
- **Headers**: `Authorization: Bearer JWT_TOKEN_STRING`
- **Query Parameters**:
  - `page`: 页码（默认 0）
  - `size`: 每页数量（默认 20）
  - `role`: 角色过滤（可选）
- **Response**:
```json
{
  "code": 200,
  "data": {
    "content": [
      {
        "id": 123,
        "username": "user1",
        "role": "USER",
        "orgTags": ["PRIVATE_user1", "TECH_DEPT"],
        "createdAt": "2025-01-01T10:00:00Z"
      }
    ],
    "totalElements": 100,
    "totalPages": 5,
    "number": 0,
    "size": 20
  }
}
```

## 二、组织标签模块

### 1. 创建组织标签（管理员）

- **URL**: `/api/v1/admin/org-tags`
- **Method**: POST
- **Headers**: `Authorization: Bearer JWT_TOKEN_STRING`
- **Request Body**:
```json
{
  "tagId": "TECH_DEPT",
  "name": "技术部",
  "description": "技术部门组织标签",
  "parentTag": "COMPANY"  // 可选
}
```
- **Response**:
```json
{
  "code": 200,
  "message": "Organization tag created successfully"
}
```

### 2. 获取组织标签列表

- **URL**: `/api/v1/org-tags`
- **Method**: GET
- **Headers**: `Authorization: Bearer JWT_TOKEN_STRING`
- **Response**:
```json
{
  "code": 200,
  "data": [
    {
      "tagId": "COMPANY",
      "name": "公司",
      "description": "公司级别标签",
      "parentTag": null,
      "createdAt": "2025-01-01T10:00:00Z"
    },
    {
      "tagId": "TECH_DEPT",
      "name": "技术部",
      "description": "技术部门组织标签",
      "parentTag": "COMPANY",
      "createdAt": "2025-01-02T10:00:00Z"
    }
  ]
}
```

### 3. 为用户分配组织标签（管理员）

- **URL**: `/api/v1/admin/users/{userId}/org-tags`
- **Method**: POST
- **Headers**: `Authorization: Bearer JWT_TOKEN_STRING`
- **Request Body**:
```json
{
  "orgTags": ["PRIVATE_user1", "TECH_DEPT", "BACKEND_TEAM"]
}
```
- **Response**:
```json
{
  "code": 200,
  "message": "Organization tags assigned successfully"
}
```

## 三、文件上传模块

### 1. 上传文件

- **URL**: `/api/v1/upload`
- **Method**: POST
- **Headers**: 
  - `Authorization: Bearer JWT_TOKEN_STRING`
  - `Content-Type: multipart/form-data`
- **Request Body**:
```
file: <binary>
orgTags: "TECH_DEPT,BACKEND_TEAM"  // 可选，默认为用户主组织
```
- **Response**:
```json
{
  "code": 200,
  "message": "File uploaded successfully",
  "data": {
    "documentId": 456,
    "filename": "技术文档.pdf",
    "fileSize": 1048576,
    "fileType": "pdf",
    "parseStatus": "PENDING"
  }
}
```

### 2. 分片上传初始化

- **URL**: `/api/v1/upload/init`
- **Method**: POST
- **Headers**: `Authorization: Bearer JWT_TOKEN_STRING`
- **Request Body**:
```json
{
  "filename": "大文件.pdf",
  "fileSize": 104857600,
  "chunkSize": 5242880,  // 5MB
  "totalChunks": 20
}
```
- **Response**:
```json
{
  "code": 200,
  "data": {
    "uploadId": "upload_789",
    "chunkSize": 5242880
  }
}
```

### 3. 上传分片

- **URL**: `/api/v1/upload/chunk`
- **Method**: POST
- **Headers**: 
  - `Authorization: Bearer JWT_TOKEN_STRING`
  - `Content-Type: multipart/form-data`
- **Request Body**:
```
uploadId: "upload_789"
chunkIndex: 0
chunk: <binary>
```
- **Response**:
```json
{
  "code": 200,
  "message": "Chunk uploaded successfully",
  "data": {
    "uploadedChunks": 1,
    "totalChunks": 20
  }
}
```

### 4. 完成分片上传

- **URL**: `/api/v1/upload/complete`
- **Method**: POST
- **Headers**: `Authorization: Bearer JWT_TOKEN_STRING`
- **Request Body**:
```json
{
  "uploadId": "upload_789",
  "filename": "大文件.pdf",
  "orgTags": ["TECH_DEPT"]
}
```
- **Response**:
```json
{
  "code": 200,
  "message": "File upload completed",
  "data": {
    "documentId": 456,
    "filename": "大文件.pdf"
  }
}
```

### 5. 获取文档列表

- **URL**: `/api/v1/documents`
- **Method**: GET
- **Headers**: `Authorization: Bearer JWT_TOKEN_STRING`
- **Query Parameters**:
  - `page`: 页码（默认 0）
  - `size`: 每页数量（默认 20）
  - `orgTag`: 组织标签过滤（可选）
- **Response**:
```json
{
  "code": 200,
  "data": {
    "content": [
      {
        "id": 456,
        "filename": "技术文档.pdf",
        "fileSize": 1048576,
        "fileType": "pdf",
        "orgTags": ["TECH_DEPT"],
        "parseStatus": "SUCCESS",
        "vectorStatus": "SUCCESS",
        "createdAt": "2025-01-01T10:00:00Z"
      }
    ],
    "totalElements": 50,
    "totalPages": 3,
    "number": 0,
    "size": 20
  }
}
```

### 6. 删除文档

- **URL**: `/api/v1/documents/{documentId}`
- **Method**: DELETE
- **Headers**: `Authorization: Bearer JWT_TOKEN_STRING`
- **Response**:
```json
{
  "code": 200,
  "message": "Document deleted successfully"
}
```

## 四、知识库检索模块

### 1. 混合检索

- **URL**: `/api/v1/search/hybrid`
- **Method**: POST
- **Headers**: `Authorization: Bearer JWT_TOKEN_STRING`
- **Request Body**:
```json
{
  "query": "什么是 RAG？",
  "topK": 5,
  "orgTags": ["TECH_DEPT"]  // 可选，默认为用户所有组织标签
}
```
- **Response**:
```json
{
  "code": 200,
  "data": {
    "results": [
      {
        "chunkId": 789,
        "documentId": 456,
        "filename": "技术文档.pdf",
        "chunkText": "RAG（Retrieval-Augmented Generation）是检索增强生成...",
        "score": 0.95,
        "pageNumber": 3,
        "chunkIndex": 5
      }
    ],
    "totalResults": 5
  }
}
```

## 五、聊天助手模块

### 1. 创建会话

- **URL**: `/api/v1/conversations`
- **Method**: POST
- **Headers**: `Authorization: Bearer JWT_TOKEN_STRING`
- **Request Body**:
```json
{
  "title": "新对话"  // 可选
}
```
- **Response**:
```json
{
  "code": 200,
  "data": {
    "conversationId": "conv_123",
    "title": "新对话",
    "createdAt": "2025-01-01T10:00:00Z",
    "expiresAt": "2025-01-08T10:00:00Z"
  }
}
```

### 2. 获取会话列表

- **URL**: `/api/v1/conversations`
- **Method**: GET
- **Headers**: `Authorization: Bearer JWT_TOKEN_STRING`
- **Query Parameters**:
  - `page`: 页码（默认 0）
  - `size`: 每页数量（默认 20）
- **Response**:
```json
{
  "code": 200,
  "data": {
    "content": [
      {
        "conversationId": "conv_123",
        "title": "关于 RAG 的讨论",
        "createdAt": "2025-01-01T10:00:00Z",
        "updatedAt": "2025-01-01T11:00:00Z"
      }
    ],
    "totalElements": 10,
    "totalPages": 1
  }
}
```

### 3. 获取会话历史

- **URL**: `/api/v1/conversations/{conversationId}/history`
- **Method**: GET
- **Headers**: `Authorization: Bearer JWT_TOKEN_STRING`
- **Response**:
```json
{
  "code": 200,
  "data": {
    "conversationId": "conv_123",
    "messages": [
      {
        "role": "user",
        "content": "什么是 RAG？",
        "timestamp": "2025-01-01T10:00:00Z"
      },
      {
        "role": "assistant",
        "content": "RAG 是检索增强生成...",
        "timestamp": "2025-01-01T10:00:05Z"
      }
    ]
  }
}
```

### 4. 删除会话

- **URL**: `/api/v1/conversations/{conversationId}`
- **Method**: DELETE
- **Headers**: `Authorization: Bearer JWT_TOKEN_STRING`
- **Response**:
```json
{
  "code": 200,
  "message": "Conversation deleted successfully"
}
```

### 5. WebSocket 连接

- **URL**: `ws://localhost:8080/chat`
- **Protocol**: STOMP over WebSocket
- **Headers**: `Authorization: Bearer JWT_TOKEN_STRING`

**发送消息**：
- **Destination**: `/app/chat`
- **Payload**:
```json
{
  "conversationId": "conv_123",
  "message": "什么是 RAG？"
}
```

**接收消息**：
- **Destination**: `/user/queue/messages`
- **Payload**:
```json
{
  "type": "chunk",  // chunk | complete | error
  "content": "RAG 是检索增强生成...",
  "conversationId": "conv_123",
  "timestamp": "2025-01-01T10:00:05Z"
}
```

## 六、通用响应格式

### 成功响应

```json
{
  "code": 200,
  "message": "Success",
  "data": { ... }
}
```

### 错误响应

```json
{
  "code": 400,  // 400: 请求错误, 401: 未认证, 403: 无权限, 404: 未找到, 500: 服务器错误
  "message": "Error message",
  "error": "Detailed error description"
}
```

## 七、认证与授权

### JWT Token 格式

**Header**:
```
Authorization: Bearer <JWT_TOKEN>
```

**Token 结构**:
```json
{
  "sub": "username",
  "userId": 123,
  "role": "USER",
  "orgTags": ["PRIVATE_user1", "TECH_DEPT"],
  "primaryOrg": "PRIVATE_user1",
  "iat": 1609459200,
  "exp": 1609545600
}
```

### 权限控制

| 接口路径 | 所需角色 | 说明 |
| --- | --- | --- |
| `/api/v1/users/register` | 无 | 公开接口 |
| `/api/v1/users/login` | 无 | 公开接口 |
| `/api/v1/users/me` | USER, ADMIN | 认证用户 |
| `/api/v1/upload/**` | USER, ADMIN | 认证用户 |
| `/api/v1/search/**` | USER, ADMIN | 认证用户 |
| `/api/v1/conversations/**` | USER, ADMIN | 认证用户 |
| `/api/v1/admin/**` | ADMIN | 仅管理员 |

## 八、速率限制

| 接口类型 | 限制 |
| --- | --- |
| 登录接口 | 5 次/分钟 |
| 文件上传 | 10 次/分钟 |
| 检索接口 | 30 次/分钟 |
| 聊天接口 | 20 次/分钟 |

超出限制返回：
```json
{
  "code": 429,
  "message": "Too many requests"
}
```
