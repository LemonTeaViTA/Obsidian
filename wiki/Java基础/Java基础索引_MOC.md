# Java 基础篇 MOC (Map of Content)

这里是 Java 基础语法的知识索引地图。Java 是一切生态的地基，这部分涉及从宏观概念、基础语法，到面向对象设计、类型系统等核心基石。

由于基础知识繁杂，我们采用了**主题聚类（Concept-Cluster）**的方式进行了合理的分拆，构建起一张清晰的从点到面的知识网。

---

## 🗺️ 知识体系导航

建议按照以下顺序进行系统学习和复习：

 Java 概述与宏观运行
了解 Java 跨平台的本质、“一次编译处处运行”、JDK/JRE/JVM的区别，以及常用的 Java 应用领域。
👉 **[[Java 概述]]**

 基础语法篇
深究 Java 处理数据的基础方式：基本数据类型、字面量与操作符、自动类型工厂。掌握各种语法糖、关键字（& 与 && 的区别等）的特性与使用。
👉 **[[基础语法]]**

 面向对象核心构建
**Java 世界观的根本。**
死磕三大特性（封装、继承、多态），理解重载与重写、抽象类与接口。深入搞懂构造方法的执行顺序与 this/super 引用。
👉 **[[面向对象]]**

 String 字符串设计哲学
字符串是 Java 中使用最频繁的对象结构。理解不可变的哲学设计、字符串驻留池、不可变带来的开销，以及 StringBuilder 与 StringBuffer 之间的使用博弈。
👉 **[[String]]**

 包装类与 Object 体系
理解为什么要有包装类的机制和极其经典的缓存池机制。死磕 Object 自带的常用方法以及 `==` 和 `equals()`, `hashCode()` 的协同机制。
👉 **[[包装类与 Object]]**

 异常处理与 I/O 体系
理解 Exception 和 Error 的派生关系；掌握各种 I/O 流的分类与阻塞 IO 机制。
👉 **[[异常与 IO]]**

 注解、反射与 Java 8
框架和高度解耦的实现基石。掌握注解系统、反射的执行性能表现，以及 Java 8 带来的强大特性（Lambda 和 Stream API）。
👉 **[[注解、反射与 Java 8]]**

---

> **学习笔记提示**：
> 基础语法的学习不仅仅是记忆规则，重点在于体会 Java 面向对象设计和内存流转的**哲学**（如：为什么 String 设计成不可变？为什么装箱使用了缓存？）。带着“为什么”学习基础，才能更好地衔接深入 JVM 和并发的进阶体系。
